# vectorworks-plugin-column-under-mark

指定したレイヤ・クラスの柱・小屋束を検索し、その平面位置に記号 (○・× 等) を
描く VectorWorks の**プラグインオブジェクト**です。適当な点に配置すると、対象の
構造材 (構造用途が柱または小屋束) を探して各柱位置に記号を描画します。柱が編集
されればオブジェクトのリセットで記号の位置・数が追随して常に最新になるため、
床伏図・小屋伏図と柱の食い違いを防げます。

構造用途に応じて記号を描き分けます。記号スタイル (`MarkStyle` パラメータ) を
選べます。

- **平面記号** (伏図記号・既定): 柱は × (2 本の線分)、小屋束は ○ (円)。挿入点
  を中心に**指定サイズ** (`MarkSize`) で描きます。
- **断面記号**: 柱は × (2 本の線分)、小屋束は / (対角線 1 本)。柱・小屋束の
  **実断面** (見え掛かりの外接矩形) に合わせて描くため、指定サイズではなく実際
  の断面寸法・位置に一致します。

どちらのスタイルでも柱は × のままで、小屋束の記号だけが切り替わります。

さらに、柱・小屋束の**上端の高さ範囲** (`TopHeightMin` / `TopHeightMax`) を
指定すると、上端がその範囲に入るものにのみ記号を描きます。伏図で対象としている
横架材の下にある柱だけに記号を付けたい場合に使います (下記「上端高さによる
絞り込み」参照)。

## しくみ

親プロジェクト (`vectorworks_plugin_import_ifc_homeskz`) と同様に、VectorWorks に
登録するのは**ラッパースクリプト `main.py` だけ**です。ラッパーはオブジェクトの
リセット (再描画) のたびに GitHub の `main` ブランチの最新コミットを確認し、
インストール済みのコミットと異なれば VectorWorks 設定フォルダ内の Python
Externals フォルダへ自動的に更新インストールしてから、プラグイン本体を実行
します (`main` は常にテスト済みのため、バージョン番号ではなくコミット SHA で
判定します)。本体パッケージはアーカイブの直接展開でインストールされます。
インターネットに接続できないなど更新を確認できない場合は、アップグレードを
スキップしてインストール済みのバージョンをそのまま実行します。

## VectorWorks へのプラグイン登録

このスクリプトは**プラグインオブジェクト (ポイントオブジェクト)** として登録
します。「ツール」→「プラグイン」→「プラグインマネージャ」等で新規の
ポイントオブジェクトプラグインを作成し、以下を設定してください。

1. **スクリプト**: `main.py` の内容をそのまま登録する。
2. **パラメータ** (プラグインオブジェクトの設定):
   - `TargetLayer` (テキスト) — 検索対象のデザインレイヤ名。空欄なら全レイヤ。
   - `TargetClass` (テキスト) — 検索対象のクラス名。空欄なら全クラス。
   - `MarkSize` (寸法または実数) — 記号のサイズ (mm)。0・未設定なら既定値
     (300mm) を使う。**平面記号でのみ使用**する (断面記号は実断面に合わせる
     ため無視される)。
   - `MarkStyle` (テキストまたはポップアップ) — 記号スタイル。`断面`
     (または `section`) を含む値なら断面記号 (小屋束=/、実断面に合わせる)、
     それ以外・空欄なら平面記号 (小屋束=○、指定サイズ) を使う。ポップアップの
     選択肢として「平面」「断面」を用意すると分かりやすい。
   - `TopHeightMin` (寸法または実数) — 柱・小屋束の上端の高さ (Z) の下限。
     空欄なら下限なし。
   - `TopHeightMax` (寸法または実数) — 柱・小屋束の上端の高さ (Z) の上限。
     空欄なら上限なし。

登録後、作成したプラグインオブジェクトを図面上の適当な点に配置すると、指定
した条件の柱の位置に ×、小屋束の位置に ○ (断面記号なら /) が描かれます。

### 上端高さによる絞り込み

`TopHeightMin` / `TopHeightMax` を指定すると、上端の高さ (Z) がその範囲に入る
柱・小屋束にのみ記号を描きます (境界は含みます)。柱が母屋まで伸びていたり
2 段梁を受けたりする場合に、伏図で対象としている横架材の下にある柱だけに記号を
付けたい、といった用途に使います。片側を空欄にすればその側は無制限、両方空欄
なら絞り込みません (すべて表示)。両方を指定して下限 > 上限になった場合は自動で
入れ替えます。上端の高さが取得できない場合は範囲を課さず表示します。

## 手動インストール (任意)

`main.py` を登録すれば初回リセット時に本体が自動インストールされるため、手動
インストールは任意です。オフライン環境などで手動インストールする場合は、この
リポジトリをクローンまたはダウンロードし、リポジトリのルートで以下を実行します。

**macOS**
```bash
pip install --target "$HOME/Library/Application Support/Vectorworks/2025/Python Externals" .
```

**Windows（コマンドプロンプト）**
```bat
pip install --target "%APPDATA%\Nemetschek\Vectorworks\2025\Python Externals" .
```

> VectorWorks のバージョンが異なる場合は `2025` の部分を実際のバージョン番号に置き換えてください。

Python Externals フォルダは VectorWorks が自動的に `sys.path` に追加するため、インストール後は追加の設定なしにパッケージを参照できます。

## 対象とする柱

親プロジェクトが配置する柱・小屋束は VectorWorks の構造材ツール
(`StructuralMember`) で描かれ、構造用途 (`StructuralUse`) が **柱="4"**・
**小屋束="5"** になります。本プラグインは指定レイヤ・クラスの構造材のうち構造
用途がこのいずれかであるものを対象に記号を描きます。
